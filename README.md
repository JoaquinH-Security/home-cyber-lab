# Home Cyber Lab

This repository documents the setup and development of my personal cybersecurity home lab.

The goal of this lab is to simulate real-world security environments where I can safely practice cybersecurity tools, techniques, and defensive strategies.

## Lab Environment

The home lab will include virtual machines and security tools used to simulate network activity and security testing.

## Technologies Used

• VirtualBox  
• Kali Linux  
• Linux CLI  
• Networking tools  
• Security testing tools  

## Lab Objectives

• Build a virtual cybersecurity lab environment  
• Practice network scanning and enumeration  
• Capture and analyze network traffic  
• Perform vulnerability assessments  
• Understand attack and defense techniques  

## Future Lab Experiments

- Kali Linux installation
- Virtual machine network configuration
- Nmap network scanning exercises
- Wireshark packet analysis
- Vulnerability scanning practice
